Connect to Firebase

One-time setup. Paste your Firebase project config below to enable real-time sync across all devices.

  1. Go to console.firebase.google.com and sign in with your Google account.
  2. Click Add project → name it (e.g. lkc-accounting) → Continue → Create project.
  3. In the project dashboard, click the Web icon (</>) → Register app → copy the firebaseConfig object.
  4. In the left menu go to Build → Firestore Database → Create database → Start in test mode → Choose a region → Enable.
  5. Go to Build → Authentication → Get started → Enable Anonymous sign-in → Save.
  6. Paste your config values below and click Connect.
